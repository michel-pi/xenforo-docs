◆ OPTION_IS_AUTOMATED

const OPTION_IS_AUTOMATED = 'isAutomated'

Meta-option for discussion message submissions that are via automated means. Some dynamic checks are skipped (eg, message length and flood check), and other data is ignored (eg, IP address).
