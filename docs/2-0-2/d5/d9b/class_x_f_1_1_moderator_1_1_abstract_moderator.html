◆ getContentTitles()

getContentTitles ( array  $ids)
abstract

Gets the title of multiple pieces of content in this content type. The return should be an array keyed by matching keys of the IDs param. Note that an ID value may occur multiple times.

Note that the title may be more than just the title, if necessary. It may include other necessary disambiguation, such as the content type ("Forum - Name").

Returns
