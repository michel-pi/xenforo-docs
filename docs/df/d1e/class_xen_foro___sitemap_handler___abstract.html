getData ( array  $entry)

Gets the sitemap data for an entry. Can either return an array with keys: loc (required, canonical URL), lastmod (unix timestamp, last modification time), priority (0.0-1.0, higher value more important), changefreq (daily/weekly/etc value), image (array with sub-options for image sitemap) Or may return an array of multiple such arrays.
