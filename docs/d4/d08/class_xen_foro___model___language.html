◆ getAllLanguagesAsFlattenedTree()

getAllLanguagesAsFlattenedTree (   $baseDepth = 0)

Gets a list of languages in the form of a flattened tree. The return is an array containing all languages and their related info. Each language additionally includes a "depth" element that repesents the depth from the (implicit) master. Children of the master have a depth 0, unless $baseDepth is overridden.

Parameters
integer$baseDepthStarting depth value.
Returns
array Format: [language id] => (array) language info, including depth
