static mapMerge ( array  $first,
array  $second 
)
static

This is a simplified version of a function similar to array_merge_recursive. It is designed to recursively merge associative arrays (maps). If each array shares a key, that key is recursed and the child keys are merged.

This function does not handle merging of non-associative arrays (numeric keys) as a special case.

More than 2 arguments may be passed if desired.

Parses a query string (x=y&a=b&c[]=d) into a structured array format.

Note that this can handle very long query strings, but it has problems if there are conflicting elements that split the "chunks" that are made internally. Workaround this using distinct keys for each input whenever possible.
