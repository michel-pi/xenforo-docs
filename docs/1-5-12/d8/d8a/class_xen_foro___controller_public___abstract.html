_getRequestIpConstraintCached ( array  $ipData,
  $sessionKey 
)
protected

Checks whether a IP constraint is matched. The value is cached in the session. The IP data should be an array with 2 keys:

  • version: unique identifier of cache revision
  • data: array of IPs to check (grouped by first byte)

If the version differs, the cache value is ignored.

Parameters
array$ipData
string$sessionKeyKey to write to/read from in the session
Returns
bool True if matched, false otherwise
