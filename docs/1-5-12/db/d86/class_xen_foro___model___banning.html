◆ banEmail()

banEmail (   $email)

Bans the specified email. Wildcards are allowed. If no wildcards are given, they are automatically added in logical places (no or leading @ gives wildcard prefix; no or trailing . gives wildcard suffix).

Throws exceptions on malformed input.

Parameters
string$email
Returns
boolean Returns true if the ban is inserted

◆ banIp()

banIp (   $ip)

Bans the specified IP or IP range. Ranges may have wildcards but only at the end (10.*, 192.168.1.*). The wildcard is optional; 192.168 is equivalent to 192.168.*.

Malformed input throws exceptions.

Parameters
string$ip
Returns
