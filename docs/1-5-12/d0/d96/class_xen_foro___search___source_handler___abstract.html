processConstraints ( array  $constraints,

)

Process search constraints.

Parameters
array$constraintsList of constraints: [constraint name] => limit (may be scalar or array)
XenForo_Search_DataHandler_Abstract | null$typeHandler
Returns
array Processed constraints. Names as keys, value is array with possible keys:
  • metadata - metadata value; keys: 0 = name of metadata, 1 = scalar/array allowed value(s) for metadata
  • query - constraint to limit via query; keys: 0 = table alias, 1 = field, 2 = operator, 3 = scalar/array allowed value(s). Multiple for "=" operator only. Note that the metadata and query keys are assumed to be equivalent. Engines need only use one (depending on engine details).
