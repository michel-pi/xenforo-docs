static

Determine if the given has to wait to perform a given action because of the flood check. If you only need to do this check, the static passFloodCheck() method is simpler.

Parameters
string$actionName of the action. Users may flood different actions, but not the same one.
integer | null$floodingLimitAmount of time where a user performing this action is flooding. If null, use default
integer | null$userIdUser doing the action. Guests never flood. If null, uses visitor
Returns
integer Number of seconds remaining until the flood check is passed
