◆ rebuild()

rebuild (   $position = 0,
array &  $options = array(),
&  $detailedMessage = '' 
)
abstract

Rebuilds the data as requested. If there is a large amount of data, it should only be partially rebuilt in each invocation.

If true is returned, then the rebuild is done. Otherwise, an integer should be returned. This will be passed to the next call as the position.

Parameters
integer$positionPosition to start building from.
array$optionsList of options. Can be modified and updated value will be passed to next call.
string$detailedMessageA detailed message about the progress to return.
Returns
integer|true
