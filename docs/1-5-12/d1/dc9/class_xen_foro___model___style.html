◆ getAllStylesAsFlattenedTree()

getAllStylesAsFlattenedTree (   $baseDepth = 0)

Gets a list of styles in the form of a flattened tree. The return is an array containing all styles and their related info. Each style additionally includes a "depth" element that repesents the depth from the (implicit) master. Children of the master have a depth 0, unless $baseDepth is overridden.

Parameters
integer$baseDepthStarting depth value.
Returns
array Format: [style id] => (array) style info, including depth
